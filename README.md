# 📊 Sales Analysis Dashboard (Power BI)

## 📌 Overview
This project presents an interactive **Sales Analysis Dashboard** built using **Power BI**.  
It provides insights into sales performance, customer distribution, orders, and financial metrics across different territories and product categories.

---

## 🚀 Key Features
- 📦 Total Orders, Revenue, Freight, Tax & Due tracking
- 🌍 Sales analysis by **Territory**
- 🛍️ Performance by **Product SubCategory**
- 📅 Monthly trend analysis
- 📊 Customer distribution insights
- 🔄 Dynamic filtering and interactivity

---

## 📈 Dashboard Insights

### 🔢 KPIs
- Total Orders: **1465**
- Total Amount: **30.09M**
- Total Freight: **915.97K**
- Total Due: **33.93M**
- Total Tax: **2.93M**

---

### 🌍 Territory Analysis
- Canada has the highest number of customers and sales
- Northwest and France follow in performance
- Central region shows the lowest activity

---

### 🛍️ Product Performance
- **Road Bikes** and **Mountain Bikes** are top-performing categories
- Accessories like **Helmets, Gloves, Jerseys** contribute smaller portions

---

### 📅 Monthly Trends
- Sales fluctuate across months
- Peak performance observed around **March & May**
- Lowest values around **April & November**
